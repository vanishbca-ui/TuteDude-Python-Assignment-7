Assignment 7 - PostgreSQL with Python

This project demonstrates how to perform CRUD operations using Python and PostgreSQL.

Technologies Used:

* Python
* PostgreSQL
* psycopg2

Features:

* Database connection
* Table creation
* Insert records
* View records
* Update records
* Delete records

Steps to Run:

1. Install psycopg2 using pip
2. Create a PostgreSQL database named "student_db"
3. Update database credentials in the code
4. Run the Python file
